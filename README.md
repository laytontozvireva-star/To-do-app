# 📝 TaskFlow - React Task Management App

TaskFlow is a simple and responsive task management application built with **React**. It helps users organize their daily activities by creating, managing, and tracking tasks in a clean and user-friendly interface.

---

## 📖 Overview

TaskFlow allows users to:

- Add new tasks
- Mark tasks as completed
- Delete completed or unwanted tasks
- View task completion progress
- Navigate between pages using React Router

This project was built as part of learning React fundamentals, including components, state management, events, and routing.

---

## 🚀 Features

- 🏠 Home page with a modern landing page
- 📋 Task management page
- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑 Delete tasks
- 📊 Live task completion counter

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/taskflow.git
```

Navigate into the project folder:

```bash
cd taskflow
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

The application will open at:

```
http://localhost:3000
```
